Features:
- Responsive Layout: Optimized for mobile, tablet, and desktop screens using Tailwind CSS.
- Scroll to Top Button: Convenient page navigation.
- Lightbox Modal: Enlarged image viewing.
- Hero Section: Interactive introduction with profile image, brief bio, and call-to-action buttons.
- About Section: Personal introduction, embedded affirmation video (Thewizardliz), and favorite song player (“Kalapastangan” by fitterkarma).

- Experience Section:
     - BeadsnTel – Founder & Creative Lead, with digital showcases using Carrd sites.
     - Leadership & organizational involvement in student orgs like Vox Nova, COMMITS, COMELEC, Google Developer Groups, Siklaw Esports, and KATAGA.
     - Academic & Technical Projects: Slideable gallery of web, system, and mobile app projects.

- Skills Section:
     - Highlights technical, creative, and leadership competencies.

- Contact Section:
     - Email and social media links for direct connection (using mailto for email).


Technologies Used:
- HTML5 & CSS3 – Semantic markup and styling.
- Tailwind CSS – Modern utility-first styling for responsive design.
- JavaScript – Interactive elements including:
      - Mobile menu toggling
      - Slideshow navigation for projects
      - Audio player controls
      - Lightbox modal for profile image
